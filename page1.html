Bot Commands

View

Bot Admins

!shutdown

 

Shuts the bot down.

!guildgeninvite

 

Generates an Invite Link to the server.

Moderation / Management

!prefix [new prefix]

 

Set prefix for the server

!purge [no. to purge] [optional user]

 

Purge a specified amount of messages, or a user's messages

!listbans

 

List all banned users in the server

!ban [user]

 

Ban a user from the server

!unban [user]

 

Unban a user from the server

!kick [user]

 

Kick a user from the server

!unbanall

 

Unban ALL banned users

!addmod [user]

 

Authorize server member's access to moderation commands

!delmod [user]

 

Revoke server member's access to moderation commands

Economy

!shop

 

Access the Server Shop.

!eco gamble [cash amount]

 

Gambles a fixed amount of Cash. Doubles cash if you win, loses cash if you lose.

Economy [Guild Admins]

!eco msgincome [msgs per income] [income]

 

Allow users to earn a fixed income [income] per [msgs per income] messages

!eco msgincome [msgs per income] [min income] [max income]

 

Allow users to earn an income ranged from [min income] to [max income] per [msgs per income] messages

!eco msgincome [true / false]

 

Enable or disable msgincome, the ability for users to gain income via messaging.

!eco reset [user]

 

Resets the inventory of the User. Note that all roles acquired via the shop must be manually removed from the User.

!eco resetall

 

Resets the inventory of all Users serverwide. Note that all roles acquired via the shop must be manually removed from affected User(s).

!eco cash [set / add / remove] [user] [cash amount]

 

Modifies User's Cash. Pretty self explanatory.
